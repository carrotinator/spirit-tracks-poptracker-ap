# spirit-tracks-poptracker-ap
Poptracker pack for [The Legend of Zelda: Spirit Tracks Archipelago](https://github.com/DayKat/spirit-tracks)

Current features: 
  - Key Item Tracker with Autotracking
  - Stamp Book Tracker with Autotracking
  - Treasure Collection Tracker With Autotracking
  - Manual Rail Tracker with Portals and Locks
  - Handling for complex rail connections
